# Security Policy

## Reporting a Vulnerability

At Cwt Academy, we take security seriously. If you discover a security vulnerability, please report it to us responsibly.

### How to Report

- **Email**: security@cwtacademy.com
- **Response Time**: We will respond to your report within 48 hours.
- **Resolution**: We will work to fix the vulnerability and notify you when it's resolved.

### Responsible Disclosure

We ask that you:
- Do not disclose the vulnerability publicly until we have had a chance to address it.
- Provide us with enough information to reproduce the vulnerability.
- Do not access or modify user data without explicit permission.

## Security Philosophy

- **Zero-Trust Architecture**: We assume no trust and verify every request.
- **Defense in Depth**: Multiple layers of security to protect against attacks.
- **Transparency**: We are committed to being open about our security practices.

## Bug Bounty Program

We are working on establishing a bug bounty program. In the meantime, we appreciate responsible reports and will credit you (if you wish) in our security acknowledgments.
